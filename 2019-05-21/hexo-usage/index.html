安装

Hexo是基于Node.js开发的，并使用Git管理。所以我们可以很方便的使用如下的指令安装它。

hexo会根据模版里面的信息建立新的文章，会使用新的主题文件赖生成静态文件。

配置

在_config.yml文件中修改配置

参数 描述 默认值
title 网站标题
subtitle 网站副标题
description 网站描述
author 您的名字
language 网站使用的语言
timezone 网站时区
url 网址
root 网站根目录
permalink 永久链接格式 :year/:month/:day/:title/

指令

  • 新建一个网站使用指令

  • 新建一篇文章

layout可选post， draft和page，分别存于source文件夹的不同位置。如果没有设置，则使用_config.yml中的default_layout参数代替。如果标题包含空格，使用引号括起来。

  • 生成静态文件

或者使用简写hexo g。

  • 发表草稿

  • 启动服务器

启动服务器。可以简写为hexo s。默认状况下，访问网址为：https:///localhost:4000

  • 部署

简写为hexo d。
在_config.yml文件中可以设置网站的部署地点。
